# 🌐 Digital Tools Buying Website

## 📖 Description
This is a React-based digital tools buying website where users can explore different products and add them to their cart. The project includes a dynamic cart system, product filtering, and a clean user interface based on Figma design.

---

## 🚀 Technologies Used
- ⚛️ React.js
- 🎨 Tailwind CSS
- 🌼 DaisyUI
- 🧠 JavaScript (ES6+)
- 🔔 React-Toastify
- 📦 JSON (for product data)

---

## ✨ Features
- 🛒 Add to Cart & Remove Functionality with real-time cart count update
- 🔄 Toggle between Products and Cart section dynamically
- 🔔 Toast notifications for add, remove, and checkout actions

---

## ⚙️ Key Functionalities
- 📊 Default product section view with 3-column layout
- 🧺 Cart system with total items count in navbar
- 💳 Proceed to Checkout button clears all cart items
- 📱 Fully responsive design for all devices

---

## 📦 Product Data
- 📄 6 products loaded from JSON
- 🏷️ Each product includes name, price, features, tag, and icon
